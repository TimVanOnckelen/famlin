Privacybeleid

Laatst bijgewerkt: juli 2026

Kort samengevat: Famlin is self-hosted, open-source software. De gegevens van jouw familie staan op de server van je eigen admin — niet op een server die het Famlin-project beheert. Er zit geen analytics of tracking in de app ingebouwd, en er wordt niets teruggestuurd naar de maker van het project.

1. Een Famlin-app of -server gebruiken

Een Famlin-server wordt opgezet en beheerd door wie hem voor zijn/haar familie heeft geïnstalleerd — de "admin." Die server van die admin bewaart alles wat Famlin nodig heeft om te werken: accountgegevens (naam, e-mailadres, en een wachtwoordhash of het subject-ID van je identiteitsprovider), posts, foto's, reacties, likes, favorieten, groepslidmaatschap, en push-tokens voor de mobiele app.

De maker van Famlin (Tim Van Onckelen) beheert of host geen enkele Famlin-server behalve die van zijn eigen familie, en heeft geen toegang tot gegevens van andermans installaties. Er is geen centrale Famlin-dienst waaraan installaties gegevens doorgeven.

Afhankelijk van hoe jouw admin de server heeft geconfigureerd, kunnen de volgende derde partijen betrokken zijn:

  • Je identiteitsprovider — als inloggen via OIDC/SSO gaat (Google, Microsoft, Authentik, of een andere provider die je admin heeft ingesteld), verwerkt die provider je login volgens zijn eigen privacybeleid. Famlin ziet of bewaart je wachtwoord in dat geval nooit.
  • Een e-mailprovider — als je admin SMTP heeft ingesteld, worden uitnodigings- en wachtwoordherstel-e-mails via die provider verstuurd.
  • De pushdienst van Expo — de mobiele app gebruikt Expo om pushmeldingen te versturen. Bij het versturen van een melding gaan het pushtoken van je toestel en de minimale inhoud van de melding via de servers van Expo naar de pushdiensten van Apple/Google.

Er zit geen analytics, telemetrie of crash-reporting ingebouwd in de backend, het adminpaneel of de mobiele app van Famlin.

Posts en reacties worden bij verwijdering soft-deleted (bewaard, verborgen en herstelbaar door een admin) in plaats van meteen gewist — vraag dus je admin als je wilt dat iets definitief verdwijnt. Back-ups, bewaartermijnen van logs en eventuele andere koppelingen zijn volledig de keuze van wie jouw installatie beheert; die persoon is de juiste om specifieke vragen over jouw gegevens aan te stellen.

2. Deze website bezoeken (famlin.app)

Deze marketingpagina heeft geen accounts, formulieren, cookies of eigen analytics. Twee dingen gebeuren automatisch wanneer je de pagina laadt:

  • Lettertypes (Nunito, JetBrains Mono) worden geladen via Google Fonts. Je browser doet daarvoor een rechtstreeks verzoek naar fonts.googleapis.com, en Google kan daarbij je IP-adres verwerken — zie het privacybeleid van Google.
  • De site draait op GitHub Pages, dat technische verzoekgegevens (zoals je IP-adres) kan loggen om de pagina te tonen — zie de privacyverklaring van GitHub.

3. Wijzigingen aan dit beleid

Famlin is een groeiend hobbyproject, dus deze pagina kan wijzigen naarmate functies veranderen. Belangrijke updates worden hier weergegeven met een nieuwe datum "laatst bijgewerkt."

4. Contact

Famlin is een hobbyproject van Tim Van Onckelen, geen bedrijf — er is geen apart privacy-e-mailadres. Vragen over dit beleid of het project stel je best via GitHub Discussions of door een issue te openen. Vragen over een specifieke Famlin-installatie en de gegevens daarin richt je aan de admin van die installatie.
