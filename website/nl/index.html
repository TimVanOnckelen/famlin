In drie stappen live

Famlin bestaat uit één backend-container plus Postgres, aangestuurd door één Docker Compose-bestand. Installeer het op een thuisserver, een NAS of een VPS — jij houdt volledige controle over de data van je familie.

deploy — famlin
# 1 · bestanden ophalen
$ git clone https://github.com/TimVanOnckelen/famlin.git
$ cd famlin && cp .env.example .env
# 2 · start Famlin
$ docker compose -f docker-compose.yml up -d
[ok] Famlin actief op :3000
[ok] Migraties toegepast
[ok] Klaar om je familie uit te nodigen
Volledige self-hosting gids →
1
Zet de server op
Eén backend-container plus Postgres, op je eigen hardware of VPS. Zet er een reverse proxy voor met TLS en je bent live.
2
Nodig je familie uit
Stuur een uitnodigingslink vanuit het adminpaneel. Iedereen logt in met de SSO-provider van jouw keuze (Google, Microsoft, Authentik, ...) of gewoon met e-mail + wachtwoord.
3
Installeer de app
Iedereen installeert Famlin voor iOS of Android en verbindt met het adres van je server. Deel je eerste herinnering.

Self-hosted & open source

Jouw data staat op jouw server, nooit bij ons. MIT-licentie en volledig te controleren — geen leverancier, geen lock-in, geen verrassingen.

Chronologische tijdlijn

Nieuwste eerst, gesorteerd op tijd. Geen ranking, geen gemiste posts — precies zoals het hoort.

Mijlpalen die opvallen

Verjaardagen en andere grote momenten krijgen een warme accentkaart — onmogelijk te missen, nooit opdringerig.

Liken, reageren & favorieten

Een hartje, een korte reactie, of een bewaarde favoriet voor later. Warme betrokkenheid zonder like-tellers of prestatiedruk.

Jouw foto's, jouw opslag

Voeg foto's rechtstreeks toe vanuit je camera of galerij. Ze worden opgeslagen op je eigen server — volledige kwaliteit, volledig van jou.

Rustig, niet verslavend

Geen oneindige scroll, geen leesbevestigingen, geen spam aan meldingen. Open de app wanneer jij daar zin in hebt.

Jij bepaalt alles

De herinneringen van je familie blijven altijd onder jouw controle.

Omdat Famlin op je eigen server draait, staat er geen bedrijf tussen jou en je familie. Geen datamining, geen algoritme dat bepaalt wat je ziet, geen tellers die je terugtrekken. Jij host het, jij bezit het, en jij sluit de app wanneer je klaar bent.
