When the server admin has connected external media sources (an
[Immich](https://immich.app) server, a local photo folder) and linked
their albums to your groups, these endpoints let you browse those
albums and reference their photos in posts. Assets are proxied through
Famlin — clients never talk to the source directly. Each linked album
carries a `provider` id saying which source it lives on.
