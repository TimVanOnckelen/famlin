Every Famlin server exposes this REST API under `/api` — the same API the
official mobile and web apps use. With a [personal access token](/developers/api)
you can build your own integrations against your family's server.

All content is scoped by **group membership**: you can only read and write
content in groups (families) you belong to, and there is no way around that
per token. See [Using the API](/developers/api) for authentication,
conventions, and worked examples.
