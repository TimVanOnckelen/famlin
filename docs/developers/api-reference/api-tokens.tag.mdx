Self-service **personal access tokens** (`famlin_pat_...`) for scripts and
integrations. The secret is returned exactly once at creation and stored
only as a SHA-256 hash. Creating a token requires an interactive login
session — a request authenticated with another API token gets a `403`,
so a leaked token can't mint replacements for itself.
